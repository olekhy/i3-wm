# i3-wm on Ubuntu 20.04

  
##### Packets are necessary to be installed

```
arandr           \
i3lock           \
i3lock-fancy     \
lxappearance     \
rofi             \
scrot            \
sudo apt install \
viewnior         \
volumeicon-alsa
```

##### Packets are optional to be installed

```
sudo apt install \
i3blocks
```
